PHP

PHP는 서버측에서 실행되는 스크립트 언어입니다.

PHP는 서버사이드 스크립트 언어입니다. 제이쿼리와 자바스크립트는 웹 브라우저에서 내장된 인터프리트에 의해 해석되는 언어입니다. 즉, 웹브라우저에서 이용할 수 있는 프로그램입니다. 빈면에 PHP는 서버에 설치된 PHP에 의해 해석됩니다. 이러한 처리를 가능하게 해석해주는 프로그램이 PHP, JSP, ASP등이 있으며 워드프레스, 줌마, 제로보드(XE), 카페24 등의 플랫폼 역시 PHP를 기반으로 하고 있습니다.

쿠키

쿠키는 세션은 어떠한 상태를 유지하는 기술입니다. 쿠키는 사용자의 컴퓨터에 상태를 저장하며, 세션은 서버에 상태를 저장합니다.

setcookie("쿠키이름","쿠키값","유지시간","쿠키적용 경로")
    <?php
    setcookie('myCookie','cookie',time()+10000,'/');  

    if(isset($_COOKIE['myCookie'])){
        echo "cookie : o";
        echo "{$_COOKIE['myCookie']}";
    }else{
        echo "cookie : x";
    }
?>

1분은 1초가 60개 있기 때문에 time() + 60을 설정

1시간은 1분이 60개 있기 때문에 time() + (60*60)을 설정

1일은 1시간이 24개 있기 때문에 time() + (60*60*24)를 설정

    <?php
    setcookie('myCookie', null, time()-1000, '/');

    if(isset($_COOKIE['myCookie'])){
        echo "cookie : o";
    }else{
        echo "cookie : x";
    }
?>
